Features

Built for anime. Powered by CV.

🎨
Semantic Layer Groups
Colours are automatically grouped into Hair, Skin, Eyes, Clothing and Background — one Krita layer per group, not one per colour.
🖌️
Auto Brush Control
Set your brush size field once. AutoDraw types the recommended size into Krita before every pass — no manual switching.
🎯
Auto Colour Paste
Saves your Krita hex field position and pastes the exact colour before each layer. Zero manual colour picking.
⚡
Bilateral Fill Processing
Noise is removed edge-aware using bilateral filtering so K-means sees clean, sharp colour zones — not blurry gradients.
✏️
Canny Outline Pass
A final black outline pass runs Canny edge detection on the original image — clean anime-style lines drawn on top of all fills.
📐
Brush Recommendation
After processing, the app recommends a brush size based on your canvas dimensions and colour count. No guesswork.
Modes

What gets drawn

Colour Fill Passes

Each colour region is filled with dense scanline strokes. The fill spacing slider controls how solid the coverage is — from 1px (completely opaque) to 8px (light hatch). Fills happen first, outlines last.

Scanline fill Spacing 1–8px Light → Dark order
🎨

Final Outline Pass

After all fills are done, AutoDraw runs Canny edge detection on the original image and draws a black outline layer on top. High thresholds ensure only real drawn lines are traced — not internal colour transitions.

Canny edge detection Black layer on top DP simplification
✏️

Semantic Layer Groups

Instead of 16+ individual colour layers, AutoDraw groups colours into meaningful categories using HSV hue and luminance analysis. Hair, Skin, Eyes, Clothing and Background — each gets its own named Krita layer.

Hair Skin Eyes Clothing Background
🗂️

Group Editor

After processing, the palette strip becomes an interactive group editor. Each group shows its colour swatches. Right-click any swatch to reassign it to a different group before drawing starts.

Right-click to reassign Live updates No reprocessing needed
